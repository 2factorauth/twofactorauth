Charles Schwab
Advanced TFA

While Schwab supports standard SMS-based two factor authentication, advanced TFA is only supported using the Symantec's VIP Access product suite through either a Symantec physical token or the Symantec VIP Access phone/desktop app. If you have other TFA sites that also use Symantec (Fidelity for example), you can use the identical Symantec VIP configuration (Schwab does not require any custom settings).

How to Setup Advanced 2FA

  1. Do one of the following:
    • Obtain a physical token by calling Schwab Technical Support Services at +1 800-435-9196.
    • Install the respective Symantec VIP Access app: iPhone, Android, MacOS or Windows
  2. Get the Credential ID from the token or app.
  3. Login to your account at schwab.com.
  4. From the top menu panel, choose Service -> Security Center. Under "Two-step Verification", select "Always at login" and then "Security Token". You will need to provide the Credential ID supplied by your token or app.
  5. The next time you log in to the Schwab website, Schwab will prompt you for the respective Security Code from your token or app.
